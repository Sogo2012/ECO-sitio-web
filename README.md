# ECO Consultor — Sitio Web

Landing pages para ECO Consultor construidas como widgets HTML en Elementor (SiteGround).

---

## Estructura del proyecto

```
ECO-sitio-web/
│
├── flower-turbines/               ← Landing page Flower Turbines
│   ├── seccion-01-hero/
│   │   └── hero.html              ✅ Listo
│   ├── seccion-02-/               🔜 Por construir
│   └── seccion-03-/               🔜 Por construir
│
└── (otras landing pages futuras)
```

## Cómo usar

Cada archivo `.html` es un widget independiente que se pega directamente en el bloque **HTML personalizado** de Elementor.

## Tecnologías

- HTML5 / CSS3 puro
- Google Fonts: Montserrat + Dosis
- Compatible con Elementor en SiteGround
- Responsive: desktop + móvil (768px breakpoint)

## Paleta de colores

| Color | Hex | Uso |
|---|---|---|
| Verde ECO | `#66913E` | Acentos, botones, stats |
| Fondo oscuro | `#0D1E24` | Background principal |
| Blanco texto | `#FFFFFF` | Títulos |
| Blanco suave | `rgba(255,255,255,0.65)` | Subtítulos |
